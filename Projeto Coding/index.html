Média

Soma todos os números do conjunto e divide pela quantidade de elementos.

Ex: 1,4,5,6

1+4+5+6/4 = 4

Mediana

Organiza em ordem crescente os elementos e retira o número que ficou no centro

Ex: 2,6,3,5,8

Mediana = 5

Moda

Seleciona o elemento mais frequente.

Ex: 3,5,6,6,8,9

Moda = 6

Variância

é o somatório do elemento menos a média ao quadrado dividido pela quantidade de elementos

∑(Xi - X)²/n

Desvio Padrão

Este é simples, bata tirar a raiz quadrada do valor da Variância

Padronização em Z

Z = (Xi - X)/DP

Média das Médias

Z = (X - μ)/EP

Erro Padrão

EP = DP/√n
